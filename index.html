Tags em HTML

Nome da Tag

Tag Basica:

html e /html: Define a raiz de um documento HTML.

Head e /head: Contém informações para o documento.

Body e /Body: Define o corpo do documento

address e /address:Define as informações de contato do autor/proprietário de um documento.

abbr e / abbr: Define uma abreviatura ou um acrônimo.

article e /article: Define um artigo.

object e /object: Define um applet incorporado

audio e /audio: Define o conteúdo de som incorporado

style e /style: Define cor de fundo

b e /b: Define texto em negrito

cite e /cite: Define o título de uma obra

code e /code: Define um trecho de código de computador.

datalits e /datalits: Especifica uma lista de opções predefinidas para controles de entrada.

dd e /dd: Define a descrição/valor de um termo em uma lista de descrições.

dialog e /dialog: Define uma caixa de diálogo ou janela.

div e / div: Define uma seção em um documento.

figure e /figure: Especifica conteúdo autossuficiente.

form e /form: Define um formulário HTML para entrada de dados do usuário.

iframe e /iframe: Define uma janela (um quadro) em um conjunto de quadros.

igm e /igm: Define uma imagem
